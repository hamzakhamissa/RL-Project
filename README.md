# Reinforcement Learning Project

## Article Importing

This project uses a local virtual environment in `.venv/` so dependencies stay inside the project and do not affect your system Python.

Create the environment:

```bash
python3 -m venv .venv
```

Activate it:

```bash
source .venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the article collector:

```bash
python collect_ton_q4_articles.py
```

Leave the virtual environment when finished:

```bash
deactivate
```
